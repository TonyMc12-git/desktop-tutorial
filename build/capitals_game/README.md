# Capitals Game

A touch-first capital city quiz built from a curated set of countries and city pools.

## How It Works

Each round shows one country name.

You tap the capital city from 20 options that deliberately mix:
- major cities from inside that country
- capitals or major cities from nearby countries and regions

## Local Testing

Run a local static server from this folder:

```powershell
python -m http.server 8011
```

Then open:

```text
http://localhost:8011/index.html
```

## Phone Install

After deployment to GitHub Pages, open:

- `https://tonymc12-git.github.io/desktop-tutorial/capitals_game/`
